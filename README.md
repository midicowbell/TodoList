# TodoList
## 🚀 API Endpoints

Ниже представлен список доступных эндпоинтов для работы с Todo-листом:

| Метод | Endpoint | Описание | Тело запроса (JSON) | Успешный ответ |
| :--- | :--- | :--- | :--- | :--- |
| `POST` | `/tasks` | Создать новую задачу | `{"title": "...", "description": "..."}` | `201 Created` |
| `GET` | `/tasks` | Получить список всех задач | - | `200 OK` (JSON с мапой задач) |
| `GET` | `/tasks/uncompleted` | Получить список невыполненных задач | - | `200 OK` (JSON с мапой задач) |
| `GET` | `/tasks/{title}` | Получить конкретную задачу по title | - | `200 OK` (JSON объекта задачи) |
| `PATCH` | `/tasks/{title}` | Изменить статус выполнения задачи | `{"completed": true/false}` | `200 OK` (Обновленная задача) |
| `DELETE` | `/tasks/{title}` | Удалить задачу | - | `204 No Content` |

> **Примечание:** Вместо `{title}` в URL необходимо подставлять уникальное название задачи.
